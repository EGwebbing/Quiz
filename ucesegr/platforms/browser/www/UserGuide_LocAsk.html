Guide content

The following guide explains how to install and use the UCL QuizApp, a location-based mobile app. Here is the content of the guide:

- System requirements;

- How to install the app;

- How the app works.

System requirements

The app works on Android OS, while iOS and Windows OS are not supported due to the lack of paid subscriptions for activation keys. The minimum supported version of Android is 5.1.0 (further information about the Android versions supported on the Phonegap website).

The app must to be connected to the UCL VPN Cisco AnyConnect to load the points of interest, further information about how to connect to the UCL VPN can be found at the following link.

How to install the app

LocAsk is still nor present on Google Play Store, therefore, it has to be downloaded and installed manually. In order to do so, the user must allow the installation from unknown sources on his device. In most Android phones, this option is found in the security section within the settings.

The installation process needs to be initialized from the following link: https://build.phonegap.com/apps/3148274/share.

Once here there are two ways to download the app:

1- Scan the QR code provided and follow the manufacturer instructions to complete the installation process;

2- Download the .apk file on the PC and move it to a phone directory that can be accessed from the phone

How the app works

LocAsk uses the built-in hardware capability of mobile devices to track the position of the user. Depending on user position and his proximity to one or more points of interest, several actions and pop alerts can be displayed. The app is composed of two main parts: the main body and the left-side menu

- The main body

The main body contains the tools that enable the user to load the questions and answer the quiz. This part consists of:

The map: here the user can see his current position and the location of the points of interest (each point is linked to a question)

The question: this section contains the question and the related answers that appear when the user is near enough to a point of interest.

The submit button: verifies the user input answer to communicate whether or not it is correct and uploads the result on the database.

- The left-side menu

This menu can be opened by tapping on the three parallel lines icon in the top-left corner of the interface. Here the user has four options:

Load points of interest: the button loads the points of interest which are the key core of the quiz.

Enable location tracking: once clicked, the option activates the continous tracking and searchs for nearby points of interest. If the user is close enough to one of them, a pop-up window will appear alerting that there is a question to be answered below. Otherwise, if the user is too far from a point of interest another pop-up window will notify him that he is too far and that there are no points nearby.

Refresh questions: enables the user to re-load all questions allowing him to re-attempt the quiz to try find the correct answer.

Clear View: this option removes any data showed on the map. The user should only use this option if they want to start over the quiz and load points of interest again.
